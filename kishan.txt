# ✅ PART C — Movie Rating Web App (DO THIS FIRST)

We’ll create a simple **Node.js web app** that prints movie ratings.

## 📌 1. Create Project in VS Code

Create a folder:

```
movie-rating-app
```

### 📄 `app.js`

```javascript
const express = require('express');
const app = express();

const movies = [
    { name: "Inception", rating: 8.8 },
    { name: "Interstellar", rating: 8.6 },
    { name: "Avengers: Endgame", rating: 8.4 }
];

app.get('/', (req, res) => {
    let result = "<h1>Movie Ratings</h1>";
    movies.forEach(m => {
        result += `<p>${m.name} - Rating: ${m.rating}</p>`;
    });
    res.send(result);
});

app.listen(3000, () => {
    console.log("Server running on port 3000");
});
```

---

### 📄 `package.json`

```json
{
  "name": "movie-rating-app",
  "version": "1.0.0",
  "main": "app.js",
  "dependencies": {
    "express": "^4.18.2"
  }
}
```

# ✅ PART D — Kubernetes Deployment

Using Kubernetes

---

## 📌 10. Deployment YAML

### 📄 `deployment.yaml`

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: movie-app
spec:
  replicas: 2
  selector:
    matchLabels:
      app: movie-app
  template:
    metadata:
      labels:
        app: movie-app
    spec:
      containers:
      - name: movie-app
        image: yourusername/movie-app
        ports:
        - containerPort: 3000
```

---

## 📌 11. Service YAML (NodePort)

### 📄 `service.yaml`

```yaml
apiVersion: v1
kind: Service
metadata:
  name: movie-service
spec:
  type: NodePort
  selector:
    app: movie-app
  ports:
    - port: 3000
      targetPort: 3000
      nodePort: 30007
```

---

## 📌 12. Apply Config

```bash
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml
```

Using Jenkins

## 📌 15. Create Pipeline Job

* New Item → Pipeline

---

## 📌 16. Add Jenkinsfile in GitHub

### 📄 `Jenkinsfile`

```groovy
pipeline {
    agent any

    stages {

        stage('Checkout GitHub') {
            steps {
                git 'https://github.com/your-username/movie-rating-app.git'
            }
        }

        stage('Build Docker Image') {
            steps {
                sh 'docker build -t yourusername/movie-app .'
            }
        }

        stage('Push to Docker Hub') {
            steps {
                sh 'docker push yourusername/movie-app'
            }
        }

        stage('Deploy to Kubernetes') {
            steps {
                sh 'kubectl apply -f deployment.yaml'
                sh 'kubectl apply -f service.yaml'
            }
        }
    }
}
```

---
